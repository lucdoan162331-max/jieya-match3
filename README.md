# 当代打工人精神状态消消乐

手机端 H5 消消乐小游戏，把职场怨气变成游戏动力。

## 快速开始

```bash
cd game
python3 -m http.server 8080
```

手机与电脑同一 WiFi 下，浏览器打开 `http://<你的IP>:8080`。

也可直接用浏览器打开 `game/index.html`（需本地 HTTP 服务以支持 ES Module）。

## 游戏特色

### 消除元素
| 方块 | 含义 |
|------|------|
| ☕ 冰美式 | 续命神器 |
| 🥞 画的大饼 | 散发金光的发面大饼 |
| 🍳 飞来横锅 | 黑色炒菜锅 |
| 📉 PPT折线图 | 永远下坡的业绩线 |
| ✓✓ 已读不回 | 两个绿色对勾 |

### 离谱机制
- **心悸模式**：连消 3 组冰美式 → 屏幕抖动，消除加速 200%
- **甩锅大法**：把锅移到棋盘边缘 → 随机砸晕隔壁格子
- **天降大饼**：一次消除 5+ 大饼 → 老板虚影 + 会议 QTE，狂点打破

### 7 个关卡
1. 周一晨会 — 入门消除
2. 需求对齐地狱 — 消灭 PPT
3. 已读不回之夜 — 控怨气消对勾
4. 续命咖啡因 — 触发心悸模式
5. 甩锅大战 — 练习甩锅
6. 大饼围城 — 触发天降大饼
7. 终极打工夜 — 综合挑战

### 开放式结局（8 种）
通关路线和打法决定结局，没有唯一正确答案：
准时下班侠、甩锅宗师、咖啡因战神、大饼受害人、精神离职、卷王之身、佛系打工人、全能社畜……

## 操作
- **点击**选中方块，再**点击**相邻方块交换
- 或**滑动**方向直接交换
- 三个及以上同色连线即可消除

## 技术
纯 HTML5 Canvas + ES Module，无构建依赖，适配手机触摸。
